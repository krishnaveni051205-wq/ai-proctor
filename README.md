# ai-proctor\
The ai proctor is a web based app which makes sure that students donot cheat in their exams 
It uses a python tech stack and Deep learning techniques to detect cheating
OmniGuard AI Proctor monitors candidates during exams using:

- 👁️ Face & Eye Tracking
- 🧠 Head Pose & Gaze Detection
- 🎤 Audio Monitoring for suspicious sounds
- ⚠️ Real-time alerts for suspicious behavior

## 🛠️ Features

- Real-time face detection
- Eye movement tracking
- Multiple person detection
- Background noise detection
- Tab-switch detection (future scope)
- Suspicious activity logging
